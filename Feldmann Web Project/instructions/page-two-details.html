Project 1: Restaurant Menu - Completion Checklist

Task 1: Import and Apply Google Fonts

Location: Top of CSS file, body styling

What to do:

  1. Go to Google Fonts
  2. Choose TWO fonts:
    • One for headings (something bold/distinctive)
    • One for body text (something readable)
  3. Copy the import statement and paste it at the very top of your CSS file:
@import url('https://fonts.googleapis.com/css2?family=Your+Font+Here');

Apply the fonts:

body {
    font-family: 'Your Body Font', sans-serif;
}

h1, h2, h3 {
    font-family: 'Your Heading Font', serif;
}
Check: Do your headings and body text use different fonts?

Task 2: Add Color Scheme

Location: Multiple places throughout CSS

What to do:

  1. Choose a color scheme for your restaurant (3-4 colors)
    • One primary color (for headers, accents)
    • One background color (light, for menu items)
    • One text accent color
  2. Use RGB or HSL format (NOT hex codes):
/* Example using HSL */
color: hsl(25, 80%, 45%); /* Orange */
background-color: hsl(45, 100%, 96%); /* Light cream */

Places to add colors:

  • body - Add a subtle background color
  • main h1 - Add a color to the main heading
  • .menu-section h2 - Add color to section headings
  • .menu-item - Add background color to menu items
  • .menu-item h3 - Add color to item names
  • .price - Add color to prices (maybe an accent color)
Check: Does your page have a cohesive color scheme? Does text have good contrast against backgrounds?

General Tips for All Projects

Box Model Reminders

  • Padding: Space INSIDE an element (between content and border)
  • Margin: Space OUTSIDE an element (between elements)
  • Border: Line around an element
  • Use these to create breathing room and organization

Float Reminders

  • Images are already floated - don't change the float property
  • Make sure floats are cleared (already handled in starter code)
  • Text should wrap around floated images naturally

Color Tips

  • Use RGB: rgb(255, 100, 50)
  • Use HSL: hsl(15, 100%, 60%)
  • For transparency: rgba(255, 100, 50, 0.8) or hsla(15, 100%, 60%, 0.8)
  • Check contrast: dark text on light backgrounds, light text on dark backgrounds

Pseudo-Class Syntax

.selector:hover {
    /* styles */
}

.selector:active {
    /* styles */
}

Pseudo-Element Syntax

.selector::before {
    content: "Text or emoji";
    /* other styles */
}

.selector::after {
    content: "";
    /* other styles */
}

Testing Your Work

Before You Submit

  1. Open in browser - does it match the wireframe?
  2. Hover over interactive elements - do they respond?
  3. Validate CSS - no errors?
  4. Check spacing - is everything balanced?
  5. Check colors - good contrast and cohesive scheme?
  6. Check typography - readable and hierarchical?
  7. Compare to wireframe image - does layout match?

Common Issues to Check

  • Prices not aligned right
  • Badges/checkmarks not showing (forgot content property?)
  • Hover effects not working (typo in selector?)
  • Colors look bad (check contrast)
  • Spacing too tight or too loose (adjust padding/margins)
  • Images look weird (check border/shadow styling)
