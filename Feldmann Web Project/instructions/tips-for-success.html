Tips for Success

Your Guide to Creating an Outstanding Website

Essential Tips & Strategies

Time Management

  • Week 1: Complete Pages 1 and 2
  • Week 2: Complete Page 3, validate code, refine design

Getting Started

  1. Choose your project option
  2. Gather content and images first (don't code without content)
  3. Start with Page 2 (the guided practice) to learn the patterns
  4. Then build Page 1 (landing page) while ideas are fresh
  5. Finish with Page 3 (independent build) to prove your understanding

Getting Help

  • Reach out to instructor about office hours if you're stuck on Page 3
  • Review the Page 2 starter code to understand layout patterns
  • Use the class resources and notes from previous weeks
  • Check the W3C validators frequently to catch errors early

For Advanced Students

  • Add extra features to Page 1 (animations, additional sections)
  • Create additional pages beyond the required three
  • Experiment with more advanced CSS techniques
  • Focus on making your design truly exceptional

Evaluation Focus Areas

Your project will be evaluated on:

Technical Proficiency:

  • Valid, semantic HTML
  • Proper use of CSS techniques covered in class
  • Working navigation between pages
  • Clean, organized code with appropriate comments

Design Quality:

  • Visual consistency across all three pages
  • Appropriate use of color, typography, and spacing
  • Professional appearance
  • Attention to detail

Page-Specific Goals:

  • Page 1: Creativity and visual appeal
  • Page 2: Following specifications and completing all required CSS additions
  • Page 3: Building a complete page from wireframe specifications

Overall Execution:

  • Meeting all technical requirements
  • Submitting all required files properly organized
  • Following instructions and specifications

Questions?

If you have questions about the project requirements, please:

  1. Review this document thoroughly first
  2. Check the starter files and wireframes provided
  3. Attend office hours
  4. Post in the class discussion forum
