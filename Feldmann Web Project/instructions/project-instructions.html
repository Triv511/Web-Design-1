HTML/CSS Three-Page Website Project

Build Your Creative Web Development Skills

Overview

In this project, you will create a complete three-page website using HTML and CSS. You will choose ONE of three project options: Restaurant Menu, Art Gallery Showcase, or Event Poster. Each project requires three distinct pages with varying levels of guidance and independence.

Project Duration: 1-2 weeks
Estimated Time: 6-10 hours

Learning Objectives

By completing this project, you will demonstrate your ability to:

  • Create semantic, valid HTML5 page structures
  • Apply CSS styling techniques including the box model, floats, positioning, and typography
  • Use pseudo-classes and pseudo-elements for interactive and decorative effects
  • Organize and maintain consistent design across multiple linked pages
  • Work independently on creative design while following technical specifications
  • Implement layouts from wireframes and starter code

Project Structure: Three-Tier Approach

Each project consists of three pages with different levels of guidance:

Page 1: Landing Page (Creative Freedom)

You will build this page completely from scratch. This is your opportunity to be creative and make something eye-catching and engaging. Minimal technical requirements are specified—focus on making a strong first impression.

Page 2: Complex Layout Page (Guided Practice)

For this page, you will receive:

  • Complete HTML structure
  • Starter CSS with basic layout
  • A wireframe/mockup showing what the finished page should look like

Your job is to complete the CSS by adding colors, fonts, spacing, hover effects, and other finishing touches according to the specifications.

Page 3: Independent Build (Structured Independence)

For this page, you will receive:

  • A wireframe/mockup image showing the desired layout
  • A list of required content elements

You must build both the HTML structure and all CSS from scratch to match the wireframe. This tests whether you've learned the patterns from Page 2.
