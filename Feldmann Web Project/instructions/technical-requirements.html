Technical Requirements

All Projects Must Meet These Standards

HTML Requirements

Code Quality & Structure

  • Valid HTML5 (must pass W3C HTML validator)
  • Semantic HTML tags used appropriately (header, nav, main, section, article, footer, etc.)
  • All three pages must be linked together with working navigation
  • Images must include alt text for accessibility

CSS Requirements

Code Standards

  • Valid CSS3 (must pass W3C CSS validator)
  • External stylesheet only (no inline styles)
  • Organized CSS with comments explaining major sections
  • Consistent styling across all three pages (navigation, header, footer should look the same)

Required CSS Techniques

Must demonstrate these techniques somewhere in your project:

Box Model: margins, padding, borders
Layout: Float and clear usage
Positioning: At least one technique (relative/absolute)
Pseudo-classes: :hover, :focus, :active, etc.
Pseudo-elements: ::before or ::after
Colors: RGB or HSL color usage
Typography: font-family, font-size, line-height, etc.
Web Fonts: At least one Google Font imported and used

Design Requirements

Visual Standards

  • Designed for desktop viewing (1000-1200px width)
  • Consistent color scheme throughout all pages
  • Consistent typography throughout all pages
  • Professional appearance with attention to spacing and alignment
  • No broken images or links

Submission Requirements

What to Submit:

Submit a compressed folder (ZIP file) containing:

  • All three HTML files
  • CSS folder with your stylesheet
  • Images folder with all images used
  • A README.txt file with:
    • Which project option you chose
    • Any special instructions for viewing your site
    • Citations for any images used (especially for Art Gallery project)

File Naming Requirements:

  • Main pages must be named: index.html (in root folder), and two other descriptive names in pages folder based on your project
    • Restaurant: pages/menu.html, pages/about.html
    • Art Gallery: pages/gallery.html, pages/artist.html
    • Event: pages/schedule.html, pages/registration.html
  • CSS file: styles.css (in css folder)
  • Images: Use descriptive names (no spaces, use hyphens: artist-portrait.jpg)

Before Submitting - Complete This Checklist:

Validate your HTML at W3C HTML validator
Validate your CSS at W3C CSS validator
Test all navigation links
Check that all images display correctly
Review that all three pages have consistent navigation and styling
